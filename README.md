# ⚡ Akash — Portfolio

A premium, futuristic developer portfolio inspired by cyberpunk interfaces and hacker aesthetics. Built with smooth animations, immersive interactions, and a modern UI to showcase projects and skills.

> **Live Demo:** https://akashhhportfolio.vercel.app

---

## 🚀 Tech Stack

* Next.js 16 (App Router)
* React 19
* TypeScript
* Tailwind CSS v4
* Framer Motion
* Lucide React

---

## ✨ Features

* 🖥️ Hacker-inspired boot sequence with animated loading screen
* 🎯 Custom interactive cursor with multiple hover states
* 🌌 Dynamic cyberpunk background with particles, grids, scan waves, and lighting effects
* ⚡ Smooth Framer Motion animations throughout the website
* ⌨️ Typewriter hero section with system initialization effects
* 🧭 Floating glassmorphism navigation with active section tracking
* 📊 Animated skills and interactive project cards
* 📅 Modern timeline for experience and education
* 🏆 Dedicated certifications and achievements section
* 💻 Terminal-style contact form
* 📱 Fully responsive across mobile, tablet, and desktop
* ♿ Accessibility-friendly with reduced-motion support

## 🛠️ Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

---

## 🎨 Customization

The portfolio is designed to be easily customizable.

You can update:

* Personal information
* Projects
* Skills
* Experience
* Education
* Certifications
* Social links
* Theme colors
* Fonts

without modifying the overall layout and animations.

---

## 🚀 Deployment

Deploy instantly using **Vercel**.

```bash
git add .
git commit -m "Update portfolio"
git push
```

Every push to the `main` branch automatically triggers a new deployment.
